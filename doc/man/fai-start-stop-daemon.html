NAME

fai-start-stop-damon − a fake of the start-stop-daemon command

SYNOPSIS

fai-start-stop-damon

DESCRIPTION

This commands returns the same exit code as start-stop-daemon(8), but does not start any daemons.

SEE ALSO

This program is part of FAI (Fully Automatic Installation). See the FAI manual for more information on how to use ftar. The FAI homepage is http://fai-project.org.
