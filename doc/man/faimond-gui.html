NAME

faimond-gui − fai monitor daemon, graphical interface

SYNOPSIS

faimond-gui FILE

DESCRIPTION

faimond-gui Read faimond(8) messages from FILE and create a graphical table using PerlTk. Normally you call it this way:

faimond | faimond-gui -

SEE ALSO

faimond(8)

This program is part of FAI (Fully Automatic Installation). See the FAI manual for more information on how to use faimond. The FAI homepage is http://fai-project.org.
