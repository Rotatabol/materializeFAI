Installation from CD, DVD or USB stick

Perform a FAI installation from CD-ROM without setting up an install server. FAI-CD puts all data needed for a fully automatic installation on a CD. No network is needed and you can install the whole system from CD.

FAI CD Multi distribution

  • This CD image can install Debian 6, CentOS 6 and Scientific Linux Cern 6 from CD
  • You can select from simple installation without any X11 or including a desktop
  • FAI CD Multi ISO image, md5sum
  • FAI 4.1, kernel 3.2, amd64, size 686MB
  • For Debian it uses the package mirror on the CD, for CentOS and SLC it connects to the fastest network mirror
  • After the installation the account demo with the password fai is available.
  • The root password is also fai.

Short description for the FAI-CD

  • The system will be installed to the first disk. All data on the disk will be deleted.
  • All log files from the installation can be found in /var/log/fai/localhost/last/
  • After the installation the account demo with the password fai is available.
  • The root password is also fai.
